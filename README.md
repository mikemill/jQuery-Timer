**jQuery Timer**

A simple plugin that is designed to fire off an event during pauses in the user's typing.

-------------------------------------------------------------
**Example usage:**

```js
$('#somerinput').timer({
	callback: function(timer)
	{
		$.post('validate.php', {
			code: timer.$elem.val(),
		}, function(data, textStatus, jqXHR)
		{
			alert(data.validate ? 'Valid' : 'Invalid');
		}, 'json');
	}
});
```
